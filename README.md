## Install

* Make Directory which install this repository
* git clone this repository

```
$ mkdir ~/my-config
$ cd ~/my-config
$ git clone git@github.com:shingo-nakanishi/git-config.git
```

### Make symbolic link

```
$ ln -s gitconfig ~/.gitconfig
$ ln -s gitignore ~/.gitignore
```

### Edit ~/.gitconfig
set your name and email
```
[user]
	name = 
	email = 
```
